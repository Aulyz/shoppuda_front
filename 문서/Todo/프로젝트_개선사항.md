## 2. 쿠폰 발행 시스템 구현 계획

### 2.1 쿠폰 모델 설계
```python
# 필요한 모델 구조
- Coupon: 쿠폰 정보
  - code: 쿠폰 코드
  - name: 쿠폰 이름
  - discount_type: 할인 타입 (정액/정률)
  - discount_value: 할인 값
  - min_order_amount: 최소 주문 금액
  - max_discount: 최대 할인 금액
  - valid_from: 유효 시작일
  - valid_to: 유효 종료일
  - usage_limit: 전체 사용 제한
  - usage_per_user: 사용자별 사용 제한
  - is_active: 활성화 여부

- CouponUsage: 쿠폰 사용 내역
  - coupon: 쿠폰
  - user: 사용자
  - order: 주문
  - used_at: 사용일시
```
